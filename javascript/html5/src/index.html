Rules

Break the hidden code by entering guesses and interpreting the exact and misplaced feedback.

Goal

Find the secret combination within the configured attempt limit. A guess is valid only when all slots are filled.

Feedback

Green counts symbols in the correct position. Amber counts symbols that exist in the secret but are placed in the wrong slot.

Configurable Options
  • Secret length can be 4 or 5.
  • Available symbol values can range from 6 to 10.
  • Values can be shown as numbers, colors, letters, or symbols.
  • Maximum attempts can be 8, 10, 12, or unlimited.
Outcomes

Winning triggers a celebratory bounce, while losing reveals the secret and tilts the last guess like falling cards.

Options

Any change is applied to the current game board after leaving the options page. New Game starts a fresh secret.
